# Technology Stack

## Core Framework
- **Next.js 16 (App Router):** The foundation for our server-first, performance-oriented frontend architecture.
- **React 19:** Utilizing the latest React features and Server Components.

## Language
- **TypeScript:** Ensuring type safety and better developer experience across the codebase.

## Styling and UI
- **Tailwind CSS:** For rapid, mobile-first styling and utility-based design.
- **shadcn/ui (Radix UI):** Providing accessible, unstyled base components that we customize via Tailwind.

## Application Logic
- **NextAuth.js v5:** Secure authentication and session management integrated with the Django backend.
- **TanStack Query (React Query):** Managing client-side data fetching, caching, and state for dynamic interactions like flight searches.
- **next-intl:** Powering the multilingual capabilities of the application.

## Backend Integration
- **Django API:** The external source of truth for travel data, bookings, and user management.
